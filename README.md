# AAA Project

A luxury minimalist agency landing page created to test the **Antigravity GitHub MCP Server** integration.

## Motivation

This project serves as a verification test for bridging the MCP server to GitHub using Docker.

## Features

- **Luxury Minimalist Design**: High-contrast, typography-focused layout.
- **Dynamic Sizing**: Responsive design using modern CSS.
- **MCP Verified**: Successfully pushed via the Antigravity toolkit.

## Stack

- HTML5 / Vanilla CSS
- Tailwind CSS (via CDN)
- Bodoni Moda & Jost Google Fonts
